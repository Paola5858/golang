# Introdução ao Go

Primeiro contato com a linguagem: declaração de variáveis, tipos de dados e constantes.

## Conceitos

- Declaração de variáveis com `var`
- Inferência de tipo
- Short declaration (`:=`)
- Constantes com `const`
- Tipos básicos: `string`, `float32`, `int`

## Executar

```bash
go run main.go
```

# 💻 Login Glamouroso com GoLang

Sistema de autenticação simples com muito brilho e estilo!

---

## 🚀 Como executar

### Compile com

```bash
go run main.go
````

---

## 🧠 Conceitos aplicados

- Entrada de dados com `fmt.Scan`
- Condicional `if/else`
- Verificação de login e senha
- Impressão divertida com emojis 💖

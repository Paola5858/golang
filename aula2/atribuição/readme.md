# ✨ Aula de Atribuição em Go ❤️

## Código de exemplo

```go
import "fmt"

func main() {
    var x int = 10
    x += 5
    fmt.Println("O valor de x é:", x)
}
```

## 💬 O que está rolando aqui?

A variável `x` começa com o valor `10`.
Com o operador `+=`, somamos `5` ao valor atual de `x`.

Ou seja:

```go
x = x + 5
```

🟡 Resultado: `15`

## 💻 Saída esperada no terminal

```plaintext
O valor de x é: 15
```
